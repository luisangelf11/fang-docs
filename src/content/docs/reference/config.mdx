En este apartado te dejaremos la configuración recomendada para crear proyectos con Fang Js, seguido de los scripts del package.json.

# Configuración de tsconfig.json

```json
{
  "compilerOptions": {
    /* Layout del Proyecto */
    "rootDir": "./src",
    "outDir": "./dist",

    /* Entorno y Resolución */
    "target": "ES2023",
    "module": "nodenext",
    "moduleResolution": "nodenext",
    "allowJs": true,
    "moduleDetection": "auto",
    "types": ["node"],
    "esModuleInterop": true,
    "skipLibCheck": true,

    /* Tipado Estricto */
    "strict": true,
    "noUncheckedIndexedAccess": true,
    "exactOptionalPropertyTypes": true,
    "isolatedModules": true,

    /* Outputs y Debugging */
    "sourceMap": true,
    "declaration": true,
    "declarationMap": true,

    /* Extras y Frameworks */
    "jsx": "react-jsx",
    "experimentalDecorators": true,
    "verbatimModuleSyntax": true,
    "noUncheckedSideEffectImports": true
  },
  "include": ["src/**/*"],
  "exclude": ["node_modules", "dist"]
}
```
